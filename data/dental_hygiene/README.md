# 🦷 dental_hygiene/

치과위생사 국가시험 원본 PDF 보관 폴더.

## 파일 명명 규칙

PDF 파일을 이 폴더에 넣을 때 다음 형식을 권장합니다.

```
YYYY년도 제NN회 치과위생사 국가시험 1교시.pdf
YYYY년도 제NN회 치과위생사 국가시험 2교시.pdf
YYYY년도 제NN회 치과위생사 국가시험 최종답안.pdf
```

## 시험 구조 (참고)

치과위생사 국가시험은 보통 다음과 같이 구성됩니다.

| 교시 | 주요 과목 |
| :---: | --- |
| 1교시 | 의료관계법규, 구강해부학, 치아형태학, 구강조직발생학, 구강병리학, 구강생리학, 구강미생물학 |
| 2교시 | 지역사회구강보건학, 구강보건교육학, 예방치과학, 치면세마학, 치과방사선학 |
| 3교시 | 임상치위생처치학(치주, 임상), 의료인 윤리 |
| 실기 | 치면세마 등 (별도 시행) |

## 주의

- 모든 PDF는 **저작권 보호**를 위해 `.gitignore`로 Git 추적에서 제외됩니다 (`data/dental_hygiene/*.pdf`).
- 본 폴더는 `README.md`만 Git에 추적됩니다.
- 학습용 시드 데이터는 `apps/api/scripts/seed_data/dental_hygiene.py`에 큐레이션되어 있습니다 (60문항).

## 향후 OCR 파이프라인

PDF를 자동으로 문제은행에 적재하는 작업은 Day 9 (KB 적재) 단계에서 정교화될 예정입니다.
현재는 `apps/api/scripts/ocr_nursing_pdfs.py`에 placeholder + TODO만 존재합니다.
