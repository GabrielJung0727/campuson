# 🩺 nursing/

간호사 국가시험 원본 PDF 보관 폴더.

## 파일 명명 규칙

```
YYYY년도 제NN회 간호사 국가시험 N교시.pdf
YYYY년도 제NN회 간호사 국가시험 1~3교시 최종답안.pdf
```

## 현재 보관 파일 (예시)

- `2026년도 제66회 간호사 국가시험 1교시.pdf`
- `2026년도 제66회 간호사 국가시험 2교시.pdf`
- `2026년도 제66회 간호사 국가시험 3교시.pdf`
- `2026년도 제66회 간호사 국가시험 1~3교시 최종답안.pdf`

## 시험 구조 (제66회 기준)

| 교시 | 주요 과목 |
| :---: | --- |
| 1교시 | 성인간호학, 모성간호학 |
| 2교시 | 아동간호학, 지역사회간호학, 정신간호학 |
| 3교시 | 간호관리학, 기본간호학, 보건의약관계법규 |

## 주의

- 모든 PDF는 **저작권 보호**를 위해 `.gitignore`로 Git 추적에서 제외됩니다.
- 학습용 시드 데이터는 `apps/api/scripts/seed_data/nursing.py`에 별도 큐레이션되어 있습니다.
