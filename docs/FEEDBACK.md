# CampusON 피드백 수집 채널

## 알파 테스트 안내

### 테스트 참여자
- 교수 1명 + 학생 5명 (간호학과 우선)
- 기간: 2주 (배포 후 시작)

### 테스트 시나리오
1. **회원가입 → 진단 테스트** (학과별 30문항)
2. **문제 풀이** (추천 문제 10~20문항)
3. **AI 채팅** (자유 질문 5~10개)
4. **오답노트 확인** + 재풀이
5. **학습 리포트 확인**
6. 교수: 학과 대시보드에서 학생 현황 확인

## 피드백 방법

### 1. GitHub Issues (개발 이슈)
- Repository: `https://github.com/<your-org>/campuson`
- 버그 리포트: Issue template 사용
- 기능 요청: Feature request template

### 2. Google Form (사용자 경험)
**학생용 설문**
- 전체 만족도 (1~5)
- 가장 유용한 기능
- 개선이 필요한 기능
- AI 해설 정확도 (1~5)
- 오답 분류 정확도 (1~5)
- 추천 문제 적절성 (1~5)
- 자유 의견

**교수용 설문**
- 대시보드 유용성 (1~5)
- 학생 학습 현황 파악 용이성 (1~5)
- AI 답변 품질에 대한 의견
- 추가 필요 기능
- 자유 의견

### 3. 직접 피드백 (Notion)
